About Robotic Simulator Choreonoid

Choreonoid is the integrated open source GUI software for robots. It has a kinetics simulation and a choreographic action function as a standard equipment. Also it is also available to put another function.
It has also an aspect as the GUI application development framework for robots.

Details about Choreonoid as a simulator

Joint movement Spin, direct operation, simple crawler
Sensor spots Force-torque sensor, gyro, acceleration sensor, camera(including RGB-D), range finder
Light Simple light with OpenGL
Simulation format Closed link feature
Shape representation Polygonal representation
Describing method Robot and environment model are described by VRML97 or COLLADA
Association RTM control system software is available
Other No simulation functions for deformation, under-water, flying

RT middleware

●It is the development software base to promote the efficiency of robot system formulation.
●It provides a component framework standardized internationally by OMG. (OpenRTM is the name of implementation.)
●It accepts any OS, language, central or distributed system flexibly.



Benefits of RT component

Reusability improvement It is available to reuse for various systems.
Diversity of option It is available to try some components with the same function.
Flexibility improvement It is available to build various systems with exchange of component format.
Reliability improvement The reliability improves to test by every component unit.
Robustness improvement

It is hard to spill over into the whole as the system is divided by component
